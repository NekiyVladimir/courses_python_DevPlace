1. Как изменить значение элемента списка?
list[index] = value

Ответ: Значение элемента списка можно изменить по индексу. Индекс значения, которое хотим поменять можно найти:
       list.index('значение') - индекс подставляем в: list[index] = 'значение на которое хотим опменять'.

4. Что выведет данный код

	a = [2, 4, 8]
	print(a[::-2])

Ответ: Выведет новый список элемент (срез). По условию будут выбираться элементы со всего списка с шагом -2 (начиная с
с конца списка с первого элемента и через один элемент до конца списка).

список. Два самых простых способа.

6. Как отсортировать список в порядке убывания?
